# ggg

`ggg`는 **good, great, gorgeous**의 약자이며,  
날씨/기후 데이터를 바탕으로 여행 의사결정을 돕는 서비스 문서 저장소입니다.

현재 저장소는 기획/스펙 중심 문서 단계이며, 앱 구현은 이후 단계에서 진행됩니다.
