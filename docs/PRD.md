# ggg — Product Requirements Document (PRD)

> **버전**: v2.1  
> **기준 목업**: climate-app-v3.html  
> **작성일**: 2025년 4월  
> **업데이트**: 구독 가격 구조 개편, 무료/유료 기능 비교표, 모드 개념 재정의 (레이어 방식), 장소 추천 + 주변 탭 신규, 네비게이션 구조 개편 (하단 탭 5개 + 햄버거 사이드 패널), 실시간 날씨 동 단위 GPS  
> **서비스 정의**: "날씨가 아니라 여행을 예측합니다" — 역사 기후 데이터 + 실시간 날씨 + 여행 콘텐츠를 통합한 B2C 여행 특화 날씨 플랫폼

---

## 1. 서비스 개요

### 1-1. 핵심 문제 정의

**문제 1 — 기후 정보 인프라의 부재 ("언제, 무엇이 좋은지")**
- 여행 계획 핵심 시점(1개월 이상 전)에 활용 가능한 기후 예측 정보가 없음
- 현행 날씨 앱은 10일 이내 단기 예보에 집중, 역사 기후 기반 여행 시기 판단 불가
- 여행자의 69%가 날씨를 여행 결정 요인으로 고려하나 이를 지원하는 서비스가 시장에 전무
- 특정 여행지·활동에 최적화된 기후 시기를 객관적 데이터로 확인할 수 없음

**문제 2 — 비수기·비인기 지역 콘텐츠의 구조적 공백 ("어디서 무엇을")**
- 여행 플랫폼과 미디어 콘텐츠 생산이 성수기·인기 명소에 집중
- 비수기에 즐길 수 있는 목적지와 활동 콘텐츠 자체가 여행자에게 도달하지 않음
- OTA·여행 플랫폼의 노출 알고리즘이 예약 수 기반으로 작동, 비수기 지역의 선순환 불가
- 여행자는 새로운 선택지를 탐색할 수단이 없어 익숙한 성수기·인기 지역을 반복 선택

> **핵심 원칙**: 성수기가 형성된 데는 이유가 있다. 강릉 8월 = 해수욕, 강릉 10월 = 트레킹·사진. 같은 곳도 목적이 다르면 가야 할 때가 달라진다. ggg는 성수기를 부정하지 않고, 여행 목적에 맞는 최적 시기와 장소를 데이터로 제시한다.

### 1-2. 소셜 미션

> **"기후 데이터로 여행 수요를 재분배하여, 오버투어리즘을 완화하고 소멸위기 지역 경제를 살린다"**

- SDG 11: 지속가능한 도시와 공동체 → 소멸위기 지역 방문 예약 비율 ≥20%
- SDG 13: 기후변화 대응 → 기후변화 인식 콘텐츠 월간 공유 수 ≥50,000건
- SDG 8: 양질의 일자리와 경제성장 → 비수기 지역 소상공인 매출 기여

### 1-3. 수익 구조

| 수익원 | 설명 | 도입 시점 |
|---|---|---|
| **디스플레이 광고** | 날씨 데이터 사이 식당·액티비티 배너. 날씨·위치 기반 초로컬 타겟팅. eCPM $8~13 | Phase 1 |
| **날씨 기반 마케팅 알림** | 소상공인이 직접 날씨 조건 설정 → 해당 날씨 시 사용자에게 마케팅 메시지 발송 | Phase 1 |
| **프리미엄 구독** | 최초 가입 ₩1,900/월. 월 5회 이상 방문 유지 시 ₩1,900 유지, 미만 시 ₩3,900으로 자동 조정. 방문 수 기반 행동 설계로 충성 유저 리텐션 유도 | Phase 1 |
| **여행 제휴 커미션** | OTA 숙박 15~30%, 액티비티 5~15% 맥락형 추천 | Phase 2 |
| **B2G 데이터** | 한국관광공사 BETTER里 연계, 소멸위기 지역 여행 수요 데이터 제공 | Phase 2 |

## 2. 사용자 정의

### 2-1. 주요 페르소나

**Persona A — 계획파 여행자 (25~35세)**
- 여행 1~6개월 전 목적지·시기 결정
- 날씨 리스크 최소화 니즈
- 핵심 기능: ggg score, 도시 비교, 숨은 황금 시즌

**Persona B — 기념일 커플 (22~30세)**
- 100일·1주년 등 기념일 날씨 예보 + 골든아워 포토스팟 니즈
- 핵심 기능: D-day 알림, 커플 모드, 날씨 아카이브, SNS 공유

**Persona C — 안전 우선 가족 (30~45세)**
- 어린이 동반 여행 시 UV·미세먼지 안전 정보 + 플랜B 니즈
- 핵심 기능: 가족 모드, 어린이 안전 종합지수, 실내 대안 추천

### 2-2. 수혜계층 (간접)

- 비수기 방문객 증가로 매출 회복을 기대하는 소멸위기 지역 숙박·음식·액티비티 사업자
- 날씨 기반 마케팅 도구를 갖게 되는 소상공인
- 오버투어리즘 완화로 삶의 질이 개선되는 인기 관광지 주민

## 3. 구독 모델 상세

### 3-1. 가격 구조

| 구분 | 가격 | 조건 |
|---|---|---|
| **무료** | ₩0 | 기본 기능, 광고 포함 |
| **프리미엄 — 활성** | ₩1,900/월 | 최초 가입 가격. 전월 방문 5회 이상 유지 시 적용 |
| **프리미엄 — 비활성** | ₩3,900/월 | 전월 방문 5회 미만 시 자동 조정 |

### 3-2. 가격 조정 로직

```
매월 1일 배치 실행 (Supabase Edge Function)
  ↓
전월 방문 횟수 집계 (user_sessions 테이블)
  ↓
  5회 이상 → ₩1,900 유지 (변동 없음)
  5회 미만 → 다음 결제 금액 ₩3,900으로 변경
            → 앱 내 사전 알림 발송
```

- 결제 처리: 토스페이먼츠 또는 아임포트(Portone) 정기결제(빌링키) API 활용
- 가격 변경은 다음 결제 시점에 적용 (소급 적용 없음)

### 3-3. 사전 알림 (UX 핵심)

방문 횟수 부족 시 알림을 통해 재방문을 유도하는 것이 이 모델의 핵심이에요.

| 시점 | 알림 내용 |
|---|---|
| 잔여 3회 필요 시 (월 말 D-5) | "이번 달 방문이 2회예요. 3번 더 오면 ₩1,900 유지!" |
| 잔여 1회 필요 시 (D-1) | "내일부터 ₩3,900이 적용돼요. 오늘 방문하면 카운트됩니다." |
| 가격 조정 당일 | "지난달 방문이 적어 이번 달부터 ₩3,900이 적용됩니다." |

## 4. 화면 구성 및 네비게이션 구조 (11개 섹션)

### 4-0. 네비게이션 아키텍처

**모드 개념 재정의**

기존의 여행자/커플/가족 분리 모드를 폐기하고, **일반 모드 기반 위에 연인·가족 레이어를 ON/OFF로 추가**하는 방식으로 전환한다.

```
서비스 기본값 = 일반 모드
                  ↓
    연인 체크 ON  →  골든아워·데이트 정보 레이어 추가
    가족 체크 ON  →  안전지수·키즈 활동 레이어 추가
    (동시 ON 가능)
```

모드 설정은 하단 탭이 아닌 **햄버거 사이드 패널 내 마이페이지**에 수납된다.

---

**하단 탭 네비게이션 (5개 고정)**

| 탭 | 아이콘 | 화면 | 설명 |
|---|---|---|---|
| 홈 | 🏠 | Home | 현재 날씨 + 모드 레이어 콘텐츠 |
| 스코어 | 🌡️ | ggg score | 목적지+시기 입력 → 기후 점수 분석 |
| 장소 추천 | 📍 | Place Finder | 목적+시기 입력 → 여행지 추천 |
| 주변 | 📌 | Nearby | 현재 위치 기반 주변 상점·카페·놀거리 |
| D-day | 📅 | D-day | 여행·기념일·소풍 D-day 공통 |

**햄버거 메뉴 (≡) → 사이드 패널 수납 항목**

| 항목 | 설명 |
|---|---|
| 모드 설정 | 연인 ON/OFF, 가족 ON/OFF 토글 |
| 숨은 황금 시즌 | 데이터 기반 비수기 발견 |
| 도시 비교 | 최대 4개 도시 기후 비교 |
| 소셜 임팩트 | 에코트립·SDG 임팩트 대시보드 |
| 마이페이지 | 구독 관리, 알림 설정, 계정 |

**PC / 태블릿**
- PC (≥1024px): 240px 풀 사이드바에 모든 항목 노출
- 태블릿 (768~1023px): 64px 아이콘 사이드바 (hover 툴팁)
- 모바일 (≤767px): 하단 탭 5개 + 우상단 햄버거(≡) → 사이드 패널

### 4-1. 홈 (Home)

**목적**: 현재 날씨 확인 + 모드 레이어별 맞춤 콘텐츠 + 서비스 핵심 가치 전달

**실시간 날씨 — 동(洞) 단위 GPS 인식**
- 기기 GPS를 통해 현재 위치를 동 단위로 인식 (예: "서울 중구 명동2가")
- 에어코리아 API 측정소 매핑: cities 테이블의 station_name 컬럼으로 가장 가까운 대기질 측정소 자동 연결
- 위치 권한 거부 시: 수동 지역 검색으로 대체
- 위치 표시: 상단 우측 위치 칩 (예: "📍 명동2가 · 지금 26°")

**모드 레이어 (사이드 패널에서 설정)**
- 기본값: 일반 모드 (모든 사용자 공통)
- 연인 체크 ON → 골든아워 품질 점수, 데이트 코스, 기념일 날씨 정보 추가 노출
- 가족 체크 ON → 어린이 안전 종합지수, UV·PM2.5 안전 등급, 키즈 활동 추천 추가 노출
- 두 레이어 동시 ON 가능

**헤로 피처 카드**
- 현재 날씨 테마에 따른 그라데이션 배경 (8종: 맑음/소나기/흐림/눈/폭염/한파/하루종일비/안개)
- 날씨 기반 한줄 인사이트 (예: "사진 찍기 완벽한 날씨예요")
- 세부 기상 정보 (골든아워 점수, 일몰 시각, 가시거리, 미세먼지 등)
- 모드별 CTA 버튼

**날씨 데이터 지표 4종**
- 현재 기온 + 체감 온도
- PM2.5 (국내: 에어코리아 / 해외: IQAir)
- 강수 확률
- ggg score (쾌적도 종합 점수)

**5일 예보 스트립**
- 날짜별 날씨 아이콘 + 기온
- D-day 날짜 강조 표시

**날씨 기반 알림 배너 (광고)**
- 날씨 조건 + 위치 기반 소상공인 마케팅 메시지
- 형식: 아이콘 + 가게명 + 날씨 맞춤 한줄 문구 + CTA
- 홈 화면에 최대 2개 삽입 (날씨 데이터 사이, 콘텐츠 카드 사이)

**숨은 황금 시즌 진입 카드**
- 모드별 다른 여행지·시기 비교 예시 표시
- 목적별 최적 시기 차이 강조 (해수욕 vs 트레킹 등)
- 클릭 시 숨은 황금 시즌 화면으로 이동

**모드별 추천 콘텐츠 3개**
- 여행자: 포토스팟, 트레킹, 야외 카페 등
- 커플: 피크닉, 노을 레스토랑, 야간 산책 등
- 가족: 어린이 시설, 실내 체험관, 캠핑 등
- 날씨 테마 변경 시 콘텐츠도 함께 변경

**에코트립 제안 배너**
- 비슷한 날씨 조건 + 소멸위기 지역 대체 목적지 추천
- ggg score + SDG 태그 표시

**소셜 임팩트 요약 바**
- 이번 달 비수기 여행 기여 참여자 수
- SDG 11 지역활성화 연계 표시

---

### 4-2. ggg score

**목적**: 특정 목적지 + 시기의 기후 적합도를 0~100점으로 정량화

**입력 영역**
- 목적지 입력 (Google Maps Autocomplete 연동)
- 여행 시기 입력 (날짜 범위 선택)
- **스코어 캘린더**: 날짜 선택 단계에서 각 날짜의 예상 ggg score를 함께 표시
  - 셀 표시: `점수(0~100)` + 등급 색상
  - 등급 라벨: 강력 추천 / 추천 / 보통 / 비추천
  - 월 전환 시 해당 월의 점수 분포를 한눈에 비교 가능

**점수 헤로 카드**
- ggg score 점수 (0~100, 대형 표시)
- 등급 레이블: 우수(75~100) / 보통(50~74) / 주의(25~49) / 불량(0~24)
- 퍼센타일 바 (역사 기후 분포 상 현재 시기의 위치)
- 추천 여부 배지 (강력 추천 / 추천 / 보통 / 비추천)

**세부 지표 4종**
- 맑은 날 확률 (역사 30년 평균 %)
- 강수 확률 (역사 30년 평균 %)
- 평균 최고기온 (°C)
- 극단 기상 리스크 (태풍·폭설·폭염 확률 %)

**월별 ggg score 바 차트**
- 12개월 월별 점수 비교
- 현재 선택 월 강조 표시
- 탭: 기후 / 강수 / 기온 전환

**역사 기후 분포 상세**
- 30년 데이터 기반 퍼센타일 분포
- 최악의 해 / 평균 / 최상의 해 비교

**목적별 ggg score** *(Phase 2)*
- 해수욕 / 트레킹 / 사진 / 미식 / 쇼핑 등 목적별 가중치 적용 점수
- 같은 목적지·시기라도 목적에 따라 점수가 달라짐

### 4-3. 숨은 황금 시즌

**목적**: 성수기 편견과 실제 기후 최적기의 차이를 데이터로 보여주고, 여행 목적에 맞는 시기를 발견하게 함

**핵심 원칙 알림 배너**
- "성수기가 형성된 이유가 있다. ggg는 목적에 맞는 최적 시기를 찾아드린다"

**목적별 비교 카드 (메인)**
- 예: 강릉 8월 (해수욕) vs 강릉 10월 (트레킹·사진)
- 각 카드: 목적 이모지 + 시기 + 대표 활동 + 기후 특성 한줄 + TCI/ggg score
- "8월 해수욕엔 8월이 맞아요 ✓" / "트레킹·사진엔 10월이 맞아요 ✓" 형태로 표현
- 성수기를 부정하지 않고 목적별 최적 시기를 안내

**숨겨진 황금 시즌 리스트**
- 형식: 지역명 + 시기 + 핵심 활동 + 기후 특성 + TCI 점수 + 태그 (비수기/절정기/추천)
- 예시
  - 제주도 11월 — 동백꽃·올레길 (동백 절정, 인파 없음, TCI 72)
  - 부산 9월 말 — 한적 해변 (수온 22°, 성수기보다 한산, TCI 74)
  - 설악산 10월 초 — 단풍 등산 (단풍 극성기, 맑은 날 78%, TCI 81)
  - 경주 봄 — 역사 유적 (UV 낮음, 선선, 벚꽃 절정)

**쿨케이션(Coolcation) 트렌드 카드**
- 서늘한 목적지 선호 증가 데이터
- 글로벌 비수기 여행 증가 트렌드

### 4-4. 장소 추천

**목적**: 여행 목적과 시기를 입력하면 기후 데이터 기반으로 최적 여행지를 추천. 모든 모드 공통 화면이나, 활성화된 레이어에 따라 추천 콘텐츠가 달라짐.

**입력 인터페이스 (공통)**
- 여행 목적 선택 (해수욕 / 트레킹 / 사진여행 / 미식 / 문화·역사 / 스키 / 힐링·온천 등 8종)
- 여행 시기 입력 (날짜 범위 또는 월 선택)
- 선택 사항: 예산 범위, 이동 거리 (당일치기 / 1박2일 / 2박3일 이상)
- **추천도 캘린더**: 시기 선택 시 날짜별 추천도를 시각화
  - 셀 표시: 추천도 배지(강력 추천/추천/보통/비추천) 또는 점수 구간 색상
  - 사용자가 날짜 범위를 드래그하면 평균 추천도와 예상 변동폭 요약 표시

**추천 결과 — 모드 레이어별 콘텐츠 차이**

| 요소 | 일반 | + 연인 레이어 ON | + 가족 레이어 ON |
|---|---|---|---|
| 장소 카드 | ggg score + 활동 태그 | 골든아워 품질 + 로맨틱 지수 추가 | 안전 점수 + 키즈 시설 여부 추가 |
| 추천 콘텐츠 | 트레킹 코스, 포토스팟, 맛집 | 일몰 포토스팟, 야경 레스토랑, 커플 코스 | 어린이 체험 시설, 안전 놀이터, 가족 식당 |
| 에코트립 배지 | 소멸위기 지역 우선 노출 | 동일 + 감성 드라이브 코스 연계 | 동일 + 안전 조건 충족 여부 표시 |

**추천 결과 화면**
- 카드 구성: 지역명 + 시기 + ggg score + 목적 달성도 + 한줄 설명 + 활동 태그 3개
- 에코트립 우선 노출: 소멸위기 지역 조건 충족 시 상단 배치 + 에코 배지
- 도시 비교 화면 연결 (최대 4개 담기)
- 숨은 황금 시즌 연계: 비수기 최적기 해당 시 황금 시즌 배지 표시

**추천 알고리즘**
```
입력: 여행 목적 + 시기 (+ 활성 레이어 필터)
  ↓
activity_weather_score 테이블 목적별 점수 조회
  ↓
best_travel_week + rain_risk_calendar 교차 검증
  ↓
목적 달성도 = activity_score × 0.6 + climate_score × 0.4
  ↓
소멸위기 지역 가중치 +5점
  ↓
레이어 필터 적용
  - 가족 ON: UV > 7 또는 PM2.5 > 35 지역 자동 하위 배치
  - 연인 ON: 골든아워 품질 점수 추가 정렬 요소로 반영
  ↓
상위 10개 목적지 반환
```

**무료 vs 프리미엄**
- 무료: 월 3회 조회, 기본 추천 상위 5개
- 프리미엄: 무제한 조회, 전체 10개 + 상세 필터 (예산·거리) + AI 분석 코멘트

### 4-5. 주변 (Nearby) — 신규

**목적**: 현재 위치 기반으로 날씨에 맞는 주변 상점·카페·음식점·놀거리를 실시간 추천

**진입 조건**
- GPS 동 단위 위치 인식 필수
- 위치 권한 거부 시 이 탭 비활성화 + 권한 요청 안내

**입력 / 필터**
- 카테고리: 전체 / 카페 / 맛집 / 액티비티 / 쇼핑 / 문화
- 거리: 도보 10분 / 도보 20분 / 차량 10분
- 날씨 필터 자동 적용: 오늘 날씨 테마에 따라 추천 카테고리 우선순위 변동
  - 맑음 → 야외 액티비티·테라스 카페 우선
  - 비 → 실내 맛집·카페·문화시설 우선
  - 폭염 → 냉방 시설 우선

**추천 결과 화면**
- 카드 구성: 장소명 + 거리 + 카테고리 태그 + 날씨 매칭 한줄 코멘트
- 지도 뷰 / 리스트 뷰 전환 가능
- 광고 배너 자연 삽입: 리스트 3번째마다 소상공인 날씨 기반 마케팅 메시지 노출

**모드 레이어별 추천 차이**

| 일반 | + 연인 레이어 ON | + 가족 레이어 ON |
|---|---|---|
| 카페·맛집·액티비티 전반 | 루프탑 바, 야경 레스토랑, 커플 카페 우선 | 키즈카페, 어린이 놀이공간, 가족 식당 우선 |

**수익 연계**
- 소상공인 날씨 기반 마케팅 알림의 핵심 노출 지면
- 날씨 조건(맑음·비·기온) 설정 → 조건 맞는 날 근처 사용자에게 알림 발송

### 4-6. 도시 비교

**목적**: 동일 시기 최대 4개 도시의 ggg score 및 세부 지표를 병렬 비교

**도시 선택 영역**
- 최대 4개 도시 선택 (+ 도시 추가 버튼)
- 여행 시기 입력 (공통 적용)
- 선택된 도시 카드: 국기 이모지 + 도시명 + ggg score + 순위 배지

**상세 지표 비교 차트**
- 맑은 날 확률 / 강수 확률 / 평균 기온 / ggg score
- 각 지표별 도시 간 프로그레스 바 비교
- 선택된 도시들의 색상 범례

**AI 분석 결론 카드**
- 형식: "10월에는 [도시명]이 최선입니다"
- 근거: 맑은 날 %, 기온, 강수 확률, 타 도시 대비 장점, 비용 비교
- 색상: 파란 배경의 강조 카드

### 4-7. D-day 알림

**목적**: 여행·기념일 날짜 저장 + D-day 날씨 예보 + 과거 날씨 아카이브

**D-day 헤로 카드**
- D-day 카운트다운 (대형 숫자, 예: "D-12")
- 이벤트명 + 날짜 + 목적지
- 5일 예보 스트립 (D-day 날짜 강조 표시)
- 역사 기후 기반 "이 날짜의 맑음 확률" 안내
  - 예: "4월 28일 제주도는 과거 20년 중 14년이 맑음 (70%)"

**날씨 아카이브**
- 저장된 기념일의 실제 날씨 기록
- 형식: 이벤트명 + 날짜 + 날씨 아이콘 + 기온 + 한줄 날씨 묘사
- SNS 공유용 날씨 카드 생성 버튼

**알림 설정**
- D-30 / D-7 / D-1 알림 설정 가능
- 날씨 조건 알림 (비 예보 시 플랜B 자동 추천)

### 4-8. 커플 모드 *(Deprecated — v2.1부터 홈·장소추천·주변 화면의 연인 레이어로 통합)*

> 별도 탭 화면으로 존재하지 않음. 사이드 패널 모드 설정에서 연인 레이어를 ON하면 홈·장소추천·주변 화면에 아래 콘텐츠가 추가 노출됨.

**연인 레이어 ON 시 추가 콘텐츠**

**골든아워 헤로 카드**
- 오늘 일몰 시각 (대형 표시)
- 골든아워 품질 점수 (0~100)
- 프로그레스 바 + 구름량 + 노을 색상 예측
- 추천 포토스팟 칩 (남산타워, 한강 노을공원 등)

**날씨 지표 4종**
- 골든아워 품질 점수
- D-day 카운트다운 (D-day 화면으로 연결)
- 야외 피크닉 적합도 (체감온도·풍속·강수 종합)
- 대기 청명도 (PM2.5 + 가시거리 + 습도 종합)

**날씨 맞춤 데이트 코스 추천 3개**
- 형식: 이모지 + 장소명 + 날씨 맞춤 한줄 설명 + 날씨 매칭 점수 (%)
- 날씨 테마·모드별 동적 변경

### 4-9. 가족 모드 *(Deprecated — v2.1부터 홈·장소추천·주변 화면의 가족 레이어로 통합)*

> 별도 탭 화면으로 존재하지 않음. 사이드 패널 모드 설정에서 가족 레이어를 ON하면 홈·장소추천·주변 화면에 아래 콘텐츠가 추가 노출됨.

**가족 레이어 ON 시 추가 콘텐츠**

**어린이 안전 종합지수 헤로 카드**
- 종합 안전 점수 (0~100, 초록 배경)
- 야외 활동 권장 시간대
- 자외선 차단제 재도포 알림 설정

**안전 지표 6종**
- PM2.5 (WHO 기준 등급)
- UV 지수 (권장 SPF 표시)
- 체감 온도 (활동 적합 범위 여부)
- 강수 확률 (우산 필요 여부)
- 풍속 (야외 놀이 가능 여부)
- 가시거리 (황사 여부)

**플랜 B 실내 활동 추천**
- 다음 날 날씨 악화 예보 시 자동 표시
- 형식: 아이콘 + 시설명 + 거리/시간 + 태그 (키즈/교육/체험)
- 위치 기반 주변 실내 시설 3개

### 4-10. 소셜 임팩트

**목적**: 서비스의 소셜미션 시각화 + 사용자 참여 독려 + 에코트립 제안

**임팩트 대시보드**
- 이번 달 비수기 여행 기여 건수
- 소멸위기 지역 방문 예약 비율
- 기후변화 인식 콘텐츠 공유 수

**SDGs 연계 지표**
- SDG 11 (지역활성화): 소멸위기 지역 방문 비율 목표 대비 달성률
- SDG 13 (기후변화): 콘텐츠 공유 수
- SDG 8 (경제성장): 비수기 지역 제휴 거래액

**에코트립 추천**
- 인기 목적지 대비 ggg score가 비슷하거나 더 높은 소멸위기 지역 추천
- 예: 강릉 대신 태안, 제주 대신 거제 등
- 추천 근거: ggg score 비교 + 방문객 밀도 + SDG 태그

### 4-11. 모드 레이어별 화면 UX 차이

모드는 독립 화면이 아니라 기본 위에 레이어를 추가하는 방식이다. 하단 탭 5개는 모든 모드에서 동일하게 노출되며, 각 화면의 콘텐츠만 활성화된 레이어에 따라 달라진다.

#### 하단 탭 구성 (모드 관계없이 고정 5개)

```
🏠 홈  |  🌡️ 스코어  |  📍 장소추천  |  📌 주변  |  📅 D-day
```

#### 화면별 레이어 적용 내용

**홈 화면**

| 요소 | 기본 (일반) | + 연인 레이어 | + 가족 레이어 |
|---|---|---|---|
| 헤로 카드 | 날씨 기반 여행 인사이트 | 골든아워 품질 점수 + 데이트 코스 추가 | 어린이 안전 종합지수 추가 |
| 날씨 지표 | 기온·PM2.5·강수·ggg score | 동일 + 골든아워 점수 | 동일 + UV 지수·안전 등급 |
| 추천 콘텐츠 | 포토스팟·트레킹·야외 카페 | 동일 + 피크닉·노을 레스토랑 | 동일 + 어린이 시설·플랜B |
| 광고 배너 | 맛집·액티비티 | 커플 패키지·야경 레스토랑 추가 | 키즈 시설·가족 체험 추가 |

**ggg score 화면**

| 요소 | 기본 | + 연인 레이어 | + 가족 레이어 |
|---|---|---|---|
| 점수 표시 | 기후 쾌적도 종합 | 동일 + 골든아워 품질 점수 추가 | 동일 + 안전 지수 강조 |
| CTA | 장소 추천 / 도시 비교 | D-day 저장 버튼 추가 노출 | 플랜B 실내 대안 확인 추가 노출 |

**장소 추천 화면**

| 요소 | 기본 | + 연인 레이어 | + 가족 레이어 |
|---|---|---|---|
| 추천 카드 | ggg score + 활동 태그 | 동일 + 골든아워 유무·로맨틱 지수 | 동일 + 안전 점수·키즈 시설 여부 |
| 정렬 가중치 | 기후 쾌적도 기준 | 골든아워 품질 추가 반영 | UV·PM2.5 안전 기준 자동 필터 |

**주변 화면**

| 요소 | 기본 | + 연인 레이어 | + 가족 레이어 |
|---|---|---|---|
| 우선 카테고리 | 카페·맛집·액티비티 전반 | 루프탑·야경 레스토랑·커플 카페 우선 | 키즈카페·어린이 놀이공간·가족 식당 우선 |

**D-day 화면**

| 요소 | 기본 | + 연인 레이어 | + 가족 레이어 |
|---|---|---|---|
| 이벤트 유형 | 여행 D-day | 기념일·데이트 (100일, 1주년 등) 추가 | 소풍·방학 이벤트 타입 추가 |
| 날씨 강조 | ggg score + 맑음 확률 | 골든아워 품질 추가 | 어린이 안전 지수 + 플랜B 추가 |

**숨은 황금 시즌 (사이드 패널)**

| 요소 | 기본 | + 연인 레이어 | + 가족 레이어 |
|---|---|---|---|
| 비교 예시 | 강릉 8월(해수욕) vs 10월(트레킹) | 제주 7월 vs 11월(동백·감성) 추가 | 설악산 봄(안전 날씨) 추가 |
| 태그 기준 | 기후 쾌적도 | 로맨틱 지수 + 인파 없음 추가 | 안전 지수 + 키즈 시설 추가 |

## 9. ggg score 알고리즘

```
ggg score = Σ(지표 × 가중치) × 100

기본 구성:
  맑은 날 확률      × 0.35
  강수 확률 역수    × 0.25
  쾌적 기온 범위    × 0.20
  날씨 안정성 지수  × 0.15
  극단 기상 리스크  × 0.05 (역수)

국내 도시: KMA TCI로 보정 레이어 추가
  최종 Score = 기본 Score × 0.7 + (TCI/100) × 0.3

목적별 가중치 조정 (Phase 2):
  해수욕: 기온 가중치 ↑, 강수 가중치 ↓
  트레킹: 날씨 안정성 가중치 ↑, 기온 범위 조정
  사진: 맑은 날·골든아워 품질 가중치 ↑
  미식: 날씨 안정성·기온 범위 가중치 ↑
```
